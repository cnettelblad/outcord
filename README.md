# OutCord
![OutCord Logo](logo.png)
> Export your Discord data with ease

OutCord is a desktop application that helps you export chat logs, channels, threads, and other important data from Discord. Whether you want to archive your communities, analyze message history, or simply backup your conversations, OutCord makes it simple.

## Features

### In Development
- **Export Channel List** - Export a list of all channels from a Discord server with metadata

### Planned Features
- **Export Server/Guild Data** - Export complete server structures including roles, categories, and settings
- **Export Messages** - Save message history from channels, threads, and DMs
- **Export Attachments & Media** - Download and archive images, videos, and files from messages
- **Multiple Export Formats** - Support for CSV/Excel and Markdown formats
- **Search & Filter** - Search and filter messages before export by date range, user, or content
- **Batch Export** - Export multiple channels or servers at once
- **Custom Templates** - Customize export format and styling

## Use Cases

- **Community Archiving** - Preserve the history of your Discord communities
- **Data Analysis** - Analyze message patterns, user activity, and engagement
- **Content Migration** - Move conversations to other platforms
- **Personal Backup** - Keep a local copy of important conversations
- **Research** - Study community dynamics and communication patterns
- **Legal Compliance** - Maintain records for legal or compliance purposes
